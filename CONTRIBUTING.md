# Contributing to PhoenixSec

Thank you for your interest in contributing to PhoenixSec!

## Getting Started
1. Fork the repository
2. Install dependencies with `pip install -e ".[dev]"`
3. Create a branch for your feature or bugfix
4. Write tests for your code
5. Submit a pull request

## Code Style
We use Ruff for formatting and linting. Run `ruff check .` before committing.
